Loss functions

The loss is the single number a model tries to make small — it defines what "good" means, and training is just gradient descent on it. The two you'll use constantly are mean squared error (for regression) and cross-entropy (for classification), and each shapes learning in its own way.

MSEMAE cross-entropy−log p outliers

Regression: squared error punishes big misses

For predicting a number, the loss measures how far the prediction ŷ is from the target y. MSE uses the square of the error, (ŷ − y)²; MAE uses the absolute value, |ŷ − y|. The difference is everything near the edges: squaring means an error of 4 costs 16 while an error of 2 costs only 4 — so MSE cares far more about large misses, which makes it smooth and easy to optimise but sensitive to outliers. Classification: cross-entropy punishes confident wrongness

For classification the model outputs a probability for the correct class, and the loss is cross-entropy: −log(p), where p is the probability it assigned to the true class. The shape is the whole story. Predict the right answer with confidence (p → 1) and the loss → 0. But predict the right answer with low probability — worse, be confidently wrong (p → 0) — and −log p shoots toward infinity. Picking the right one

Match the loss to the task: MSE (or MAE if you have outliers) for regression; cross-entropy for classification, paired with a sigmoid output for binary or softmax for multi-class. Using MSE on a classifier technically works but trains poorly — its gradients are tiny exactly when the model is confidently wrong, whereas cross-entropy's gradient stays strong, pushing hard to fix the worst mistakes. The loss isn't an afterthought; it's the objective that everything else serves.

Takeaways: the loss defines "good" and is what gradient descent minimises. MSE squares errors → smooth but outlier-sensitive (regression); MAE is more robust. Cross-entropy = −log(p_true) → ~0 when confident-correct, explodes when confident-wrong (classification, with sigmoid/softmax). Match the loss to the task; cross-entropy gives a classifier far better gradients than MSE.
