The Illustrated ReAct Agent

An agent isn't magic — it's a loop: Reason → Act (call a tool) → Observe, repeat until it can answer. Step through a real trace below; the tools actually run.

agentstool use reasoning loopBYO-10 ReAct agent

The model never does arithmetic or lookups itself — it decides which tool to call, the tool returns an Observation, and that gets fed back so the model can decide the next step. The max_steps cap stops runaway loops.
