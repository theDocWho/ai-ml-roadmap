The Illustrated RAG Pipeline

Retrieval-Augmented Generation, end to end — type a question and watch it chunk → embed → retrieve → ground → generate, fully offline, no API calls.

retrievalembeddings cosine similarityBYO-9 RAG engine

An LLM only knows what's in its weights. RAG gives it fresh, private knowledge by retrieving relevant passages and pasting them into the prompt as context — so the answer is grounded in your documents and can cite its sources. Below is a tiny, real RAG engine running in your browser (a bag-of-words embedder + cosine search + a templated generator).

The pipeline
